# 运动学分析指南

## 连杆机构

### 曲柄摇杆机构（Grashof 条件）

最短杆 + 最长杆 ≤ 其余两杆之和

| 条件 | 机构类型 |
|:----|:--------|
| 最短杆固定 | 双曲柄 |
| 最短杆为连杆 | 双摇杆 |
| 最短杆为曲柄且满足 Grashof | 曲柄摇杆 |

### 滑块曲柄机构

位移: x = r·cosθ + l·cos(arcsin(r·sinθ/l))
速度: v ≈ -rω·sinθ (l >> r 时)
最大行程: 2r

## 凸轮机构

### 常用运动规律

| 规律 | 最大速度 | 最大加速度 | 冲击 |
|:----|:------|:--------|:----|
| 等速 | 1.0 | ∞ | 刚性 |
| 等加速等减速 | 2.0 | 4.0 | 柔性 |
| 简谐 | 1.57 | 4.93 | 柔性 |
| 摆线 | 2.0 | 6.28 | 无 |

## 齿轮系

### 传动比计算

- 单级: i = z_out / z_in = ω_in / ω_out
- 多级串联: i_total = i1 × i2 × ...
- 行星轮系: i = 1 + z_ring / z_sun (行星架输出)

### 效率

| 传动类型 | 效率范围 |
|:--------|:------|
| 直齿圆柱齿轮 | 98-99% |
| 斜齿轮 | 97-98% |
| 蜗轮蜗杆 | 50-90% (自锁 < 50%) |
| 谐波减速器 | 70-90% |
| 行星减速器 | 94-98% |
